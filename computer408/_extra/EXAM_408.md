    >

  * ❌07(无向图连通最少边数)
  
    > 题目要求"任何情况都连通"，即无论怎么选边去连，图都是连通的。这时候不能用 $n-1$ 的生成树连法，虽然这是最少的边，但同样 $n-1$ 条边，只要不按照生成树的连法去连，图仍然不连通(如 $4$ 个顶点，结果 $3$ 条边连了个三角形，一个顶点还是孤立的)。这时候应在非连通图最大边数的基础上加一，这样无论怎么连都是连通图，即边数为 $C^2_{n-1}+1$
    >
    > ```Text
    > 1——2
    > | /
    > 3  4
    > ```
    >
    > 如图 $4$ 个顶点 $3$ 条边，不按生成树连法连，导致图不连通

  * 09(折半查找判定树的高度)
  
    > 问折半查找的最多次数，则转换为问 $n$ 个结点的折半查找判定树有多高
  
  * ❌15(芯片字位扩展后的地址)
  
    > 用小芯片字位扩展成大芯片后，地址线低位表示小芯片的地址，高位表示片选信号，用与选择不同的芯片组。若给出一个地址，则只需比对片选位，即可确定该芯片组号。单个小芯片的低位地址都是全 $0$ 到全 $1$

  * 17(Cache，TLB，页表三者的命中情况)

  * 36(ICMP报文类型)
  * ❌38(广播风暴、*网桥)
  * ❌39(TCP滑动窗口)
  
    > 本质上乙的接收窗口大小=甲发出但还在路上(乙没确认)+甲还没发(全新待发)的量，因为只有乙收到了才能进行计算扣除，产生新的接收窗口。所以乙声明的接收窗口包含了以上两部分

  * 40(DNS递归查询)
  * ❌41(散列表、ASL)
  
    > $\text{ASL}_{成功}$ 的分母是元素的个数；$\text{ASL}_{失败}$ 的分母是模能覆盖的地址的范围

  * ❌42(数组移动)
  
    > C语言函数传数组实现
    >
    > ```C
    > void func(int arr[], int n) {
    >     arr[0] = 10;  // 和本地数组用法完全一样
    >     for (int i = 0; i < n; i++)
    >         printf("%d ", arr[i]);
    > }
    > ```
    >
    > 使用`int arr[]`看起来能像普通变量一样使用，不增加记忆负担，但本质上这是个数组指针，只不过没有显式声明出来——使用时也无需显式操作指针地址。**仅有的代价就是应该同时传入数组长度`n`的参数**
