# 应用文写作(小作文)

## 1 思路

### 1.1 总览

* 审题三事项

  a. 背景  
  b. 体裁 + 内容要求（描述）  
  c. 语域（正式/非正式）+ 格式

* 三段式简易思路

  **首段 1-2 句：**  
  a. 首给熟人：首句寒暄，二句目的；  
  b. 写给陌生人或机构：自我介绍和目的。
  
  **中间段 4-6 句：**  
  按题目要求写相应的内容，并辅以感谢、强调、对比、建议、说明等功能句完成写作。
  
  **尾段 1-2 句：**  
  （再次）致歉、感谢、期待、展望等。

### 1.2 道歉信

* 开头

  很高兴回想起在之前...
  > $\text{I'm pleased to recall the past 时间 when ...}$
  
  我写信的目的是...
  > $\text{I'm writing for the purpose of doing ...}$

* 正文

  **按题目要求写相应的内容**

  我是如此粗心，以至于我才发现/忘了...
  > $\text{I'm so careless and I did not found ...}$

* 结尾

  最后，我真诚地希望您能站在我的角度考虑一下并接受我的道歉
  > $\text{Finally, I truly hope that you can stand in my shoes and accept my apologies.}$

  再次为引发的不便向您道歉
  > $\text{Once again, I am sorry for any inconveniences caused.}$

## 2 格式

* 书信

  ```Text
  +------------------------------------------+
  | Dear xxx,                                |
  |     [开头]I am writing to ......          |
  |     [正文]............................... |
  | ........................................ |
  | ............................             |
  |     [结尾]............................... |
  | ..................                       |
  |                           Your sincerely |
  |                                  Li Ming |
  +------------------------------------------+
  ```

* 通知

  ```Text
  +------------------------------------------+
  |                 Notice                   |
  |                      Monday, December 20 |
  |     [开头]This notice is for .......      |
  |     [正文]............................... |
  | ........................................ |
  | ............................             |
  |     [结尾]............................... |
  | ..................                       |
  |                              发通知的单位 |
  +------------------------------------------+
  ```

## 3 功能句

* 强调

  > $\text{It is 狗 that 狗}$

  ***示例：***

  > $\text{It is practice that make perfect.}$

* 应该停止某事

  > $\text{It is advisable for sb. to do sth.}$ </br>
  > 停止做某事是明智的 $\Rightarrow$ 某人不应该继续做某事

* 自我介绍

  > $\text{As + 身份 + 定语从句}$ </br>
  > 作为一个(身份)，我....

  这样写的目的是使写信人与收信人双方产生联系

* 写信为了

  > $\text{I am writing [for the purpose of] doing sth. [since/because] + 从句}$ </br>
  > 我写信是为了sth.，因为...
